Several industry standard color names made available as CSS custom properties (vulgo: variables).

* [RAL paint and pigment colors](ral/), German industry standard since 1927 in 3 variants
  * RAL Classic, extended from 40 (1927) to 216 base colors with standardized names in several European languages
  * RAL Design System (1993), CIE-Lab-based, but slightly altered subsequently
  * RAL Effect (2007), 420 plain and 70 metallic hues
  * RAL Plastics (2010) equivalents for Classic and Design
* [HKS offset printing and ink colors](hks/), German industry standard since 1968 with 88 named base colors for 4 types of paper (`N` normal, `K` photo/art, `Z` newspaper, `E` continuous stationery) in CMYK
* [ISO 28178:2013 C5 SWOP colors](swop/), 1617 measured samples in CMYK
* [Printer Working Group (PWG) Media Standardized Names (MSN) Media Color Names](pwg/), established for networked printing protocols like IPP, 20 base colors, most of them in 4 variants
